# CentShift
CentShift é uma aplicação de gestão financeira pessoal que permite aos utilizadores registar, acompanhar e gerir transações, definir orçamentos e acompanhar despesas, investimentos e poupanças de forma intuitiva. A aplicação combina uma API em Python com uma interface visual em Streamlit.
## Funcionalidades Principais

- **Transações**: criar, consultar e filtrar despesas, receitas, investimentos e poupanças.  
- **Gestão de Orçamento**: aplicar modalidades automáticas de divisão do salário (ex.: 50% despesas, 30% investimentos, 20% pessoal).  
- **Relatórios e Exportação**: gerar relatórios mensais e exportar dados em CSV/JSON.  
- **Interface Visual**: dashboards interativos com gráficos, tabelas e resumo financeiro.  

---

## Tecnologias

- **Backend**: Python, FastAPI, SQLModel/SQLAlchemy, Pydantic  
- **Frontend**: Streamlit, Pandas  
- **Base de Dados**: SQLite (fácil adaptação para Postgres)  
- **Ferramentas**: Uvicorn, Pytest, Docker (opcional)
